# About

Enter string and select encrypt way sha1 or md5.

You get encrypt string.

## How to use

Enter string:

```
$ ruby encrypt.rb
Введите слово или фразу для шифрования:
> Мама мыла раму
```

Enter number of encryption method:
```
Каким способом зашифровать:
1. MD5
2. SHA1
> 1
```

You can see result:

```
Вот что получилось:
c7b73e27e80c6ccd198bd0c4c34dfbd0
```
